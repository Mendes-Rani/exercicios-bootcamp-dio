# Exercícios realizados
### Exercício 1 - NameAndAge 
Para o exercício 1 proposto, a classe **NameAndAge** é uma classe simples que representa uma pessoa com um nome e uma idade. Ela possui um construtor para inicializar os atributos e métodos para acessar e modificar esses atributos.

### Exercício 2 - AreaQuadrada
Para o exercício 2, a classe **AreaQuadrada** é uma classe que calcula a área de um quadrado. Ela possui um atributo para armazenar o lado do quadrado e um método para calcular a área com base nesse lado.

### Exercício 3 - AreaRetangulo 
Para o exercício 3, a classe **AreaRetangulo** é uma classe que calcula a área de um retângulo. Ela possui atributos para armazenar a largura e a altura do retângulo, e um método para calcular a área com base nesses atributos.

### Exercício 4 - DiferencaIdade
Para o exercício 4, a classe **DiferencaIdade** é uma classe que calcula a diferença de idade entre duas pessoas. Ela possui atributos para armazenar as idades das duas pessoas e um método para calcular a diferença de idade com base nesses atributos.
